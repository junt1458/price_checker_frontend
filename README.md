# Price checker 
Frontend repository of [Price Checker](https://github.com/YYtech15/price_checker)

TECHCAFE 2022 Winter Hackathon project

# Environment
- Node.js
- TypeScript
- Next.js
- Tailwindcss

# Commands
```
# Initial setup
$ yarn

# Start development server
$ yarn dev

# Build pages
$ yarn build

# Start production server
$ yarn start

# Lint your code
$ yarn lint
```
